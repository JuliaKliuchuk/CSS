### Run Environment

***
1. ##### Clone repo
```sh
$ git clone git@github.com:JuliaKliuchuk/CSS.git
$ cd ./CSS
```

2. ##### install
```sh
$ npm i
```


3. ##### start localhost server
```sh
$ npm start
```
